Build a complete mobile-first web app called "Mochi Mode".

IMPORTANT:
- Keep all logic clean and organized.
- Use HTML, CSS, and JavaScript.
- Use localStorage for persistence.
- Do not include PWA install banners.
- Focus on responsive mobile UI.
- Do not break logic when styling.

--------------------------------------------------
1. HOME SCREEN
--------------------------------------------------

Use a full-screen pixel art background image (user provided).

Overlay:
- Centered app logo text: "Mochi Mode 🐾"
- Small subtitle: "Focus gently. Grow daily."
- A rounded button at bottom center: "Start"

When user clicks Start:
→ Navigate to Main Dashboard screen.

--------------------------------------------------
3. POMOPAWS SESSION SCREEN
--------------------------------------------------

Features:
- Input field: "What are we focusing on?"
- Duration buttons: 10m / 25m / 50m
- Circular animated countdown timer
- Start / Cancel button
- Kitty displayed under timer

During session:
- Show rotating motivational messages if toggle enabled.
- Animate kitty subtly.
- Save session on completion:
    {
      purpose,
      minutes,
      date
    }
- Store in localStorage as "mochiSessions"

XP System:
- 1 minute = 1 XP
- Store in "mochiXP"
- Level up every 200 XP
- Show level progress bar

Streak System:
- Completing 3 sessions in one day increases streak by 1.
- Only increment once per day.
- If user misses a day → reset streak.
- Store in "mochiStreak"
- Store last streak update date.

When streak hits milestones:
3 → Common Kitty
7 → Rare Kitty
14 → Epic Kitty
30 → Legendary Kitty

Show congratulatory popup + confetti.
Store unlocked kitties in "mochiUnlockedKitties".

- Use rounded corners (border-radius: 16px+).
- Soft shadows.
- Smooth transitions.
